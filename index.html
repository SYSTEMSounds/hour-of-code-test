Orbit Sonification with P5

This is the code that will be bundled into blocks in blockly.

1. Add a Star

Place a star at the center. Choose its color and mass (which changes the size)

2. Add a Planet

Add a planet. Apply gravity from the star so that it orbits. Draw a path.

3. Change Planet's Speed

The orbits are only circular if speedFactor=1. Changing this slightly will create elliptical orbits.

4. Pluck a String

Create a line (string) and pluck it when a planet passes.

4. Play a Note

Play a musical note when a planet plucks the string.

4. Add Many Planets

Add several planets with different notes.

5. Choose a Scale

Choose notes from a certain scale. Can also add an arbitrary number of planets.
